Vendored assets for the generated YARD theme.

Fonts:

- Inter Latin and Latin Extended WOFF2 files from Google Fonts.
- Stabil Grotesk Regular, Medium, and Bold WOFF2 files from lavinmq.com.

YARD copies this directory into `doc/lavinmq/` through `.yardopts`, so the
generated API documentation does not depend on remote font or asset hosts.
